# Olá, eu sou Henderson

Estudante de Ciência da Computação, construindo uma base sólida em desenvolvimento back-end, APIs e boas práticas de código.

Atualmente estou estudando C#, .NET, SQL e fundamentos de arquitetura através de projetos práticos. A ideia é evoluir com calma, entendendo bem cada camada do que estou construindo.

## Estudando Agora

- Desenvolvimento back-end com C# e ASP.NET Core
- APIs REST, autenticação e integração com front-end
- Banco de dados relacional com SQL e MySQL
- Angular e TypeScript para consumir APIs
- Organização de projetos, versionamento e documentação

## Projeto em Destaque

**MuseuDeBugs**

Um projeto pessoal para registrar bugs encontrados durante os estudos, guardar causa, solução, linguagem, status e transformar cada erro em aprendizado documentado.

O projeto usa ASP.NET Core Web API, Entity Framework Core, MySQL, autenticação por cookie e um front Angular consumindo a API.
